Cybersecurity Awareness Chatbot 

A WPF-based Cybersecurity Awareness Chatbot developed in C# using Visual Studio 2022.

This chatbot was created to educate users about important cybersecurity topics such as:

Password safety
Phishing scams
Privacy protection
Online scams
Safe browsing

The application uses a graphical user interface (GUI), voice greeting, ASCII art, memory features, sentiment detection, and keyword recognition to create an interactive cybersecurity learning experience.
- Features
- WPF Graphical User Interface
- Voice Greeting using WAV audio
- ASCII Cybersecurity Shield Logo
- Keyword Recognition
- Random Cybersecurity Responses
- Conversation Flow
- Sentiment Detection
- User Memory Feature
- Error Handling and Input Validation
- Object-Oriented Programming Structure
- Use of Dictionaries and Lists
